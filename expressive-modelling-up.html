Expressive Modelling - Up

Description

Project made as part of the course Expressive Modelling for 3D printing of Pr. Marie-Paule Cani
JavaScript (ThreeJS)

The course consisted in learning the basis of Computer Graphics (Illuminations, shape and objetcs representations, implicit surfaces, etc.). Using what we had seen on the course, the goal of our project was to design a javascript program in which the user can model a flying house, like in the Pixar animated movie "Up", in the most intuitive manner as possible. This consisted for instance in determining the way mouse events would interact with the structure. Once the program was finished, we printed our house at the school fablab.
