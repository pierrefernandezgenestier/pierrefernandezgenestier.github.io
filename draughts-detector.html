Draughts detector

Description

Project made as part of a Computer Vision course (INF573)
C++ OpenCV

To put in in a nutshell, the goal of the program is to process an image of a chessboard and to return the picture of the chessboard seen from above and a digital representation of the game (as a matrix of -1, 0 and 1). To do so we used C++ OpenCV with methods such as Canny edge detectors, HoughLines and HoughCircles transformations, as well as clustering methods to select the right lines.
