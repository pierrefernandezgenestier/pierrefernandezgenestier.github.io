GYM Car-Racing

Description

The project was made as part of the course Advanced Topics in Artificial Intelligence (INF581) 
Python (OpenAI Gym, Keras)

I have implemented different designs of autonomous agents on the OpenAI gym Car-Racing-v0 environment, which is a random 2D top-down racing environment. The problem is very challenging since it requires computational resources, given the high state space. Here, I have compared different algorithms to solve the environment (DQN, DDQN, GAs, REINFORCE). 
The most promising one was a method that looks like DQNfD (2017 Paper from DeepMind), but other algorithms that required computer vision to extract lower dimensional features were really interesting to study.
