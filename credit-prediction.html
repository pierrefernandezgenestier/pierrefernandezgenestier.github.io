Credit Default Prediction

Description

Project done as part of Machine Learning II (MAP559)
Jupyter Notebook

In this work, I was trying to predict credit default risk, using user data such as their history, their age, other important dates, etc. 
The most important part was the data preprocessing part, which had to deal with how to interprete dates, categorical and numerical data from a csv file. Moreover, since the training dataset was imbalanced (luckily for the bank!), we had to use methods to enhance the dataset (e.g. SMOTE). To predict the label of the default credit (true or false) we used classical methods: regression logistic, neural network, svm, etc. and we assembled the results to have a more robust prediction.
